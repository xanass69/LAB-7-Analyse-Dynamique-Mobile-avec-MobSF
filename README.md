# LAB-7-Analyse-Dynamique-Mobile-avec-MobSF

## 📋 Description
Ce projet documente l'analyse dynamique de l'application volontairement vulnérable **DIVA** (Damn Insecure and Vulnerable Android App) en utilisant **MobSF** (Mobile Security Framework).

## 🎯 Objectifs
- Comprendre l'analyse dynamique d'applications Android
- Détecter des vulnérabilités en temps réel
- Maîtriser l'utilisation de MobSF avec Frida
- Intercepter et analyser le trafic HTTPS

## 🛠️ Technologies utilisées
- **MobSF** v4.5.0 - Framework d'analyse de sécurité mobile
- **DIVA** - Application Android vulnérable (13 challenges)
- **Docker** - Conteneurisation de MobSF
- **Android Emulator (AVD)** - API 30 (Android 11)
- **Frida** - Instrumentation d'applications

## 📦 Prérequis
- Docker Desktop
- Android Studio avec SDK
- Minimum 8 Go RAM
- Windows 10/11 (ou Linux/Mac)
